# Learn Fast Technology Skill

Bộ skill này giúp bạn học nhanh và áp dụng nhanh một công nghệ mới theo hướng 80/20, thực chiến.

## File trong thư mục

- `SKILL.md`: skill chính, dùng cho AI như ChatGPT, Claude, Cursor, Codex.
- `example-kafka.md`: ví dụ áp dụng skill cho Kafka.
- `quick-prompt.md`: prompt ngắn để copy nhanh.

## Cách dùng

1. Mở `quick-prompt.md`.
2. Thay `{{TECHNOLOGY}}`, `{{BACKGROUND}}`, `{{GOAL}}`, `{{TIMEFRAME}}`, `{{CURRENT_STACK}}`.
3. Dán vào AI.
4. Yêu cầu AI xuất thành markdown nếu muốn đưa vào VitePress hoặc learning portal.

## Gợi ý dùng với VitePress

Bạn có thể tạo cấu trúc:

```txt
learning/
  kafka/
    00-overview.md
    01-core-concepts.md
    02-mini-project.md
    03-debug-checklist.md
    04-production-checklist.md
  kubernetes/
  redis/
  docker/
```
